About Gna.Iata.Js Demo

This website demonstrates use of (Gna.Iata library) in regular JavaScript website (as wasm module). Instead of rewriting everything for the web, developers can write the logic once in C# and turn it into code that runs smoothly in any browser. It’s a smart way to save time, reduce complexity, and keep everything working consistently across different platforms.

  • Ssim parsing: Avionworx [wasm]
  • 3D Visualisation: Globe.GL [threejs]
  • Airport data: OurAirports
  • Random data: jpatokal /openflights
  • CSS Style:Bootstrap [css]
